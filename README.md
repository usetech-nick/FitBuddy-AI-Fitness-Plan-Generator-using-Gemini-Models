\# 💪 FitBuddy – AI Powered Fitness Plan Generator



FitBuddy is a full-stack AI-powered fitness web application built using FastAPI and Google Gemini API.  

It generates personalized 7-day workout plans based on user goals, age, weight, and intensity preferences.



---



\## 🚀 Features



\- 🧠 AI-generated 7-day workout plans

\- 🎯 Goal-based customization (Weight Loss, Muscle Gain, General Wellness)

\- 🔄 Feedback loop to regenerate plans

\- 🥗 Daily nutrition tip generation

\- 💾 SQLite database storage

\- 🎨 Modern responsive UI



---



\## 🛠️ Tech Stack



\- \*\*Backend:\*\* FastAPI

\- \*\*Frontend:\*\* HTML, CSS, JavaScript

\- \*\*Database:\*\* SQLite

\- \*\*AI Model:\*\* Google Gemini 2.5 Flash

\- \*\*ORM:\*\* SQLAlchemy
